# UCMS Bulletin
Newbulletin module for Ultimo MVC

## Requirements
* PHP 5.3
* Ultimo MVC Config
* Ultimo MVC ORM
* Ultimo MVC Form
* Ultimo MVC Phptpl
* Ultimo MVC ACL
* Ultimo MVC Translate
* Ultimo MVC Validation
* UCMS Visualiser
* UCMS Commentor

## Features
* List, add, update and delete messages
* Schedule messages in the future
* I18n
* Images
* Comments
